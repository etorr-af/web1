Webpage Structure

The overall structure of this website follows a clear and consistent layout across all pages to ensure ease of navigation and maintainability. Each page uses semantic HTML5 elements to improve readability for both developers and assistive technologies.

  • <!DOCTYPE html>: Declares the document type to ensure consistent rendering across browsers.
  • <html>, <head>, <body>: These elements define the main structure of each webpage.
  • <header>: Contains the site’s title and navigation links, repeated on each page for consistency.
  • <nav>: The navigation bar allows users to access different sections/pages of the website easily.
  • <section>: Used to group related content into logical blocks (e.g., hobbies, K-pop, games, technicalities).
  • <div>: Helps organize content within each section and apply specific styles using CSS classes.
  • <table>: Used to organize and present information in a structured and easy-to-read format.
  • <footer>: Appears at the bottom of each page with acknowledgements or reference notes.

Using this consistent structure across all pages not only improves the user experience but also makes the website easier to update and maintain.

Accessibility

This website has been designed with accessibility in mind to ensure it can be used by as many people as possible, including those with disabilities. Semantic HTML5 tags are used to provide a clear content structure, which benefits screen readers and other assistive technologies. Navigation links are consistent across pages to aid usability. Text sizes and colors have been chosen for readability and sufficient contrast. Alt attributes are provided for images to describe visuals for users with visual impairments. The layout is responsive, so it adapts to different screen sizes including mobile devices.
